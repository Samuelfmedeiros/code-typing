Desafio de digitação de código. Digite snippets de várias linguagens o mais rápido possível!

## Como Jogar
1. Um snippet de código aparece na tela
2. Digite no campo abaixo
3. Verde = certo, Vermelho = errado
4. Ao final, veja seu WPM e precisão

Linguagens: Python, JavaScript, TypeScript, React, Go, Rust, SQL

## Stack
- React 18 (CDN) · Babel Standalone · CSS puro

## Executar
```bash
git clone git@github.com:Samuelfmedeiros/code-typing.git
cd code-typing
open index.html
```
